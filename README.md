# Golf Membership App
Figma Prototype: https://www.figma.com/design/L4RHHikuwQI9su2VdE5Uff/Mobile-Development?node-id=0-1&t=52g1Vvh82TAeo3Vh-1

Golf Membership App was developed as part of RSU Hackathon 2025 to improve how golf club members manage reservations, membership benefits, and rewards in one place. The project focused on creating a simple and modern mobile experience that allows users to view their membership information, track loyalty points, redeem rewards, and manage bookings more conveniently.

For this project, I worked as a UI/UX Designer and Frontend Developer in my team. I designed the user flows and interfaces in Figma and developed the mobile application using Flutter and Dart. I also implemented both light and dark mode designs while ensuring the interface remained clean, responsive, and easy to navigate.

**Technologies:** Flutter, Dart, Figma, GitHub, Android Studio
